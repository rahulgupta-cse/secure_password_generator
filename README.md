#  Password Generator (Python CLI)

A simple and efficient **Password Generator** built using Python.
This tool helps users create strong and secure passwords based on their preferences.

---

##  Features

*  Custom password length
*  Includes uppercase & lowercase letters
*  Optional digits
*  Optional special characters
*  Fast and lightweight CLI tool
*  No external dependencies required
*  Lightweight with minimal system resource usage
*  Open-source and easy to audit
*  Suitable for personal and enterprise security needs

---

##  Technologies Used

* Python
* Random Module
* String Module

##  How to Run

1. Clone the repository:

```
git remote add origin https://github.com/rahulgupta-cse/secure_password_generator.git

```

2. Navigate to the folder:

```
cd python-password-generator
```

3. Run the program:

```
python main.py
```

---

## Example Usage

```
Enter password length: 10
Include digits? (yes/no): yes
Include special characters? (yes/no): no

Generated Password: Ab3kLm9QwX
```
